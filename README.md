## Development

### Installation

```bash
npm install
```

### Local Development

```bash
npm start
```

Starts a local development server at `http://localhost:3000`.

### Build

```bash
npm build
```

Generates static content into the `build` directory.

### Deployment

```bash
npm run deploy
```

Deploys to GitHub Pages.
